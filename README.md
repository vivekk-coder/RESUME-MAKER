# Resume Builder App

A simple Resume Builder web application built using React.js. This app allows users to create and customize their resumes easily.

## 🚀 Features
- Create professional resumes
- User-friendly interface
- Real-time preview

## 🛠️ Tech Stack
- React.js
- HTML
- CSS
- JavaScript

## 📂 Project Setup

Clone the repository:
git clone https://github.com/vivekk-coder/RESUME-MAKER.git

Go to project folder:
cd repo-name

Install dependencies:
npm install

Run the app:
npm start
